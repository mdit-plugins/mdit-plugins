# @mdit/plugin-layout

[![Version](https://img.shields.io/npm/v/@mdit/plugin-layout.svg?style=flat-square&logo=npm)](https://www.npmjs.com/package/@mdit/plugin-layout)
[![License](https://img.shields.io/npm/l/@mdit/plugin-layout.svg?style=flat-square)](https://github.com/mdit-plugins/mdit-plugins/blob/main/packages/layout/LICENSE)

A directive-based layout engine plugin for MarkdownIt supporting Flexbox, CSS Grid, and Multi-column layouts.

## Install

```bash
npm i @mdit/plugin-layout
```
