inline-rule plugin for MarkdownIt.

## Install / 安装

```bash
# pnpm
pnpm add -D @mdit/plugin-inline-rule
# npm
npm i -D @mdit/plugin-inline-rule
# yarn
yarn add -D @mdit/plugin-inline-rule
```
